# Clínica — Site institucional e landing pages

Projeto inicial para o site do Centro de Especialidades em Saúde e Bem Estar, com foco em tráfego pago, captação de leads e agendamento via WhatsApp.

## Stack inicial

- React
- Vite
- CSS puro
- Estrutura preparada para receber a logo oficial em `public/logo.png`

## Como rodar localmente

```bash
npm install
npm run dev
```

## Próximos passos

1. Enviar a logo oficial sem fundo.
2. Subir a logo em `public/logo.png`.
3. Substituir os serviços provisórios pelos serviços reais.
4. Inserir WhatsApp, endereço e horário de atendimento.
5. Configurar eventos de conversão para Google Ads, Meta Ads e WhatsApp.
